## crushist
Crowd Sourced JukeBox

###Description

Crowd-Sourced, YouTube-Powered Jukebox made with Django Framework.

###Development
This is my first Web-Dev project, so I documentation and testing are not keeping up with the project. As I better learn the framework, I will bring the project more up to standards.

2. Start virtualenv
3. Run pip install -r requirements.txt
